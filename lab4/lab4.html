Internet Technology 2021/22

Task 1

Przygotować stronę na serwerze Pascal ( katalog labs/index.html ) opracowaną w HTML5 z wykorzystaniem styli CSS umożliwiającą wyświetlenie zadań realizowanych na kolejnych zajęciach w formie jednej strony z linkami do opracowanych stron na serwerze Pascal i w chmurze obliczeniowej IBM Cloud. Strony w chmurze obliczeniowej IBM Cloud będziemy realizować od laboratorium nr. 3.

Task 2

Przygotować stronę na serwerze Pascal ( katalog labs/index.xml ) opracowaną w języku XML z wykorzystaniem styli CSS umożliwiającą wyświetlenie zadań realizowanych na kolejnych zajęciach w formie jednej strony. Obsługa przekierowanie do innych stron (linki) nie działają z wykorzystaniem tej metody formatowania dokumentu XML.
